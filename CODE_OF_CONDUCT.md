# Code of Conduct

## Our Pledge
As the maintainer of Guto, I am committed to providing a respectful and harassment-free experience for everyone who contributes to this project. 

## Standards
I expect all participants to:
* Use welcoming and inclusive language.
* Be respectful of differing viewpoints and experiences.
* Gracefully accept constructive criticism.
* Focus on what is best for the project and the community.

## Scope
This Code of Conduct applies only to online interactions within this repository (Issues, Pull Requests, and Discussions). This is a solo-led project, and all communication is handled asynchronously.

## Enforcement
If you encounter behavior that violates these standards, please report it by opening a private issue or contacting me directly via GitHub. I will review and address the situation to the best of my ability.
