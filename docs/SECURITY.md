# Security Policy

## Supported Versions
Guto is currently in its early stages. Only the latest version is actively supported.

## Reporting a Vulnerability
If you discover a security vulnerability, please do not disclose it publicly. 

* **How to report:** Use the GitHub "Report a vulnerability" feature in the Security tab of this repository.
* **Response:** As a solo maintainer, I will review and respond to reports as quickly as possible, usually within a few days.
